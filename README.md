# Envoy.Extras.Autofac
Autofac extensions for [Envoy](https://github.com/peterkneale/envoy)

## Installation

via Package Manager

```Install-Package Envoy.Extras.Autofac```

via .NET CLI: 
    
```dotnet add package Envoy.Extras.Autofac```

via Packet CLI

```paket add Envoy.Extras.Autofac```
